# Quindioflix-BD-II

QuindioFlix es una plataforma colombiana de streaming de contenido multimedia que requiere un sistema de base de datos robusto capaz de soportar su operación completa. El sistema debe gestionar el catálogo de contenidos, los usuarios y sus perfiles, el consumo de contenido, la facturación y pagos, el equipo de trabajo interno y los reportes .
